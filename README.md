# Molecular Dynamics Software based on Pure Functions in Fortran
Author: Shinji Iida

## Implimentation
* System: 
    - Face-Centered Cubic lattice, 2d and 1d simple lattice

* Potential:
    - Lenard Jones

* Periodic boundary

* Integrator:
    - velocity verlet

* Thermostat:
    - velocity rescaling
